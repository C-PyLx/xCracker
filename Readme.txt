Usages:

1) Open an terminal into xCracker directory
2) Use command ./install.sh to install into
3) Wait for install & configuration
4) Then > Use commands:
-----------------------
xCracker -t [Hash-Type] -H [Hash-Value] -w [Wordlist] -v <verbose>
xCracker -t [md5] -H e710bd5c9954d75a973406effb384095 -w [words.txt] -v
xCracker -t [sha1] -H 11f6ad8ec52a2984abaafd7c3b516503785c2072 -w [words.txt]
-----------------------
5) And wait for crack hashes :)
